1

Сократить время на рутинную проверку работоспособности ссылок и выявления опечаток в них

Решение

Скрипт на Python для автоматической проверки. При помощи регулярных выражений извлекаются ссылки и проверяются с помощью библиотеки requests. Нерабочие ссылки записываются в таблицу и проверяются вручную.

Результат

Из примерно 500–700 ссылок как нерабочие выявляются 15–20. Большинство из них содержат опечатку (ненужный пробел, пропущенная точка). Экономия времени до 3 часов на каждую книгу.

2

Выявить опечатки в названиях нерусифицированного интерфейса

Решение

Скрипт на Python с применением регулярных выражений, извлечение уникальных названий на латинице, подсчет числа их вхождений в текст и ручная проверка (единичное вхождение — возможная опечатка).

Результат

Экономия времени до 12 часов для отдельных книг соответствующей тематики и более высокое качество для остальных.

3

Создать сайт для продвижения личного бренда

Решение

Из имеющегося макета в Figma извлекаются стили и картинки. Затем пишется HTML для каждой страницы и заливается в новый репозиторий GitHub. Для помощи используется ИИ (DeepSeek).

Результат

Личный бесплатный сайт за минимум времени, дающий возможность не только поделиться своими успехами, но и оттачивать навыки (технические, аналитические и др.).
